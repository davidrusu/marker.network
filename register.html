Register Device

We need to link this app with your reMarkable tablet through the reMarkable Cloud service

https://my.remarkable.com/connect/desktop

If you don't see a one-time code after logging into your reMarkable account, You may need to click the link once more.

A note on privacy, please read:

This one time code is used to generate a "device token", this device token can be used to read and write to your reMarkable tablet. It's important that you don't share this code with anyone otherwise you'd be handing over the keys to your personal notes and thoughts.

In order for marker.network to function, we need to access the files on your tablet. To do this we store this device token on your computer. All interaction with the reMarkable cloud will happen from your computer. No one else will have access to this device token.

To publish your site, we will upload the contents of a folder on your device to marker.network. You'll be prompted before we upload anything to be sure everything is in order.

If you have any questions about any of this, please reach out to me at davidrusu.me@gmail.com

If you're curious what's going on under the hood, the source code for this app is available at https://github.com/davidrusu/marker.network
